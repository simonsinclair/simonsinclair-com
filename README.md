# simonsinclair.com

A static site built using Eleventy.

## Overview
```
/src
  /site - processed by Eleventy.
    /_includes - files included from Nunjucks templates.
      /css
      /js
      /layouts
      /partials
    /articles - articles and index.
  /assets - static assets, which are copied to /public/assets.
  /lib - To do.
```
