# simonsinclair.com

My personal website.

## Getting started

### Setup

```
git clone https://github.com/simonsinclair/simonsinclair-com.git
cd simonsinclair-com
npm install
npx playwright install
```

### Running locally

#### Development

1. `npm run dev`
2. `npm run serve`

#### Production

```
npm start
open http://localhost:8080/
```

### Running tests

```
npm test
```
